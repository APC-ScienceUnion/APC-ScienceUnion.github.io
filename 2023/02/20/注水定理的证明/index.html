注水定理的证明

问题描述

  “注水定理”解决的是信息论中的一个基本问题：以总容量最大化为目标的AWGN信道功率分配方案优化。该问题描述如下：有 KKK 个并联 AWGN 信道且噪相互声独立，噪声功率依次为 σ12,σ22,⋅⋅⋅,σK2σ^2_1,σ^2_2,··· ,σ^2_Kσ12​,σ22​,⋅⋅⋅,σK2​。总功率受限于 PPP，求出使 KKK 个并联信道功率最大化的功率分配方案。

  写做优化问题格式为：

max⁡p1,…,pK∑k=1Klog⁡2(1+pkσk2)s.t.∑k=1Kpk≤Ppk≥0,   k=1,2,…,K(1.1)\begin{aligned} &\max_{p_1, \dots , p_K}\sum^{K}_{k=1} \log_2 \left(1 + \frac{p_k}{\sigma ^2_k}\right)\\ \mathrm{s.t.}&\sum^K_{k = 1}p_k \le P\\ &p_k \ge 0, \ \ \ k = 1, 2, \dots , K\end{aligned} \tag{1.1}s.t.​p1​,…,pK​max​k=1∑K​log2​(1+σk2​pk​​)k=1∑K​pk​≤Ppk​≥0,   k=1,2,…,K​(1.1)

一个简单的证明

  在这一节中我们给出一个比较便于理解的推导。

  有的人可能觉得直接将所有功率分配到最好的（噪声功率最小的）信道里就可以，但信道容量的表达式是 Ck=log⁡2(1+pk/σk2)C_k = \log_2 (1 + p_k / σ^2_k)Ck​=log2​(1+pk​/σk2​)，随功率增加，增长的幅度越来越小，这种想法启发我们每次将功率分配到容量增长幅度最大的信道中。

  假设我们已有一个初始功率分配方案 p1(0),p2(0),…,pK(0)p^{(0)}_1 ,p^{(0)}_2 , \dots ,p^{(0)}_Kp1(0)​,p2(0)​,…,pK(0)​ ，满足：

∑k=1Kpk(0)<P,   pk(0)≥0,   k=1,2,…,K\sum^K_{k = 1}p^{(0)}_k < P, \ \ \ p^{(0)}_{k} \ge 0, \ \ \ k = 1, 2, \dots , Kk=1∑K​pk(0)​<P,   pk(0)​≥0,   k=1,2,…,K

  显然 p1(0)=p2(0)=⋯=pK(0)=0p^{(0)}_1 = p^{(0)}_2 = \dots = p^{(0)}_K = 0p1(0)​=p2(0)​=⋯=pK(0)​=0 也是一个合理的初始化方案。假设一正数 (P−∑k=1Kpk(0))≥δ>0(P - \sum^K_{k=1}p^{(0)}_k) \ge \delta > 0(P−∑k=1K​pk(0)​)≥δ>0 ，代表接下来要分配的功率。把 δ\deltaδ 功率分配到第 kkk 个信道中，则信道容量增加：

ΔC=log⁡2(1+pk(0)+δσk2)−log⁡2(1+pk(0)σk2)=log⁡2(1+δpk(0)+σk2)(2.1)\Delta C = \log_2 \left( 1 + \frac{p^{(0)}_k + \delta}{\sigma^2_k}\right) - \log_2 \left(1 + \frac{p^{(0)}_k}{\sigma^2_k}\right) = \log_2 \left(1 + \frac{\delta}{p^{(0)}_k + \sigma^2_k}\right)\tag{2.1}ΔC=log2​(1+σk2​pk(0)​+δ​)−log2​(1+σk2​pk(0)​​)=log2​(1+pk(0)​+σk2​δ​)(2.1)

  不难发现应该将功率分配到 pk(0)+σk2p^{(0)}_k + \sigma^2_kpk(0)​+σk2​ 最小的信道中。如果每次选取的 δ\deltaδ 尽可能小，则将功率完全分配后，一部分信道的功率分配满足 σk2+pk\sigma^2_k +p_kσk2​+pk​ 为定值，对于另一部分较差的（噪声功率较大的）信道，σk2σ^2_kσk2​ 大于该定值，则不分配功率，即：

pk=max⁡{0,p∗−σk2}(2.2)p_k = \max \{0, p^* - \sigma^2_k\}\tag{2.2}pk​=max{0,p∗−σk2​}(2.2)

  其中 p⋆p^⋆p⋆ 满足 ∑k=1Kpk=P\sum^K_{k = 1}p_k = P∑k=1K​pk​=P 。

  如果对于某一个分配方案若 pi+σi2>pj+σj2p_i + \sigma^2_i > p_j + \sigma^2_jpi​+σi2​>pj​+σj2​ ，我们置 δ=min⁡{pi,(pi+σi2−pj−σj2)/2}\delta = \min\{p_i, (p_i + \sigma^2_i - p_j - \sigma^2_j)/2\}δ=min{pi​,(pi​+σi2​−pj​−σj2​)/2} ，对于状态 pi(0)=pi−δ,pj(0)=pjp^{(0)}_i = p_i - \delta, p^{(0)}_j = p_jpi(0)​=pi​−δ,pj(0)​=pj​ ，根据之前的推导，把 δ\deltaδ 分配到信道 jjj 比分配给信道 iii 能获取更多的信道容量。重新分配后 pi+σi2=pj+σj2p_i + \sigma^2_i = p_j + \sigma^2_jpi​+σi2​=pj​+σj2​ 或 pi=0p_i = 0pi​=0（此时 σi2≥pj+σj2\sigma_i^2 \ge p_j + \sigma^2_jσi2​≥pj​+σj2​ ）。

一个严谨的证明

  我们重写 (1.1)(1.1)(1.1) 为：

min⁡p1,…,pK−∑k=1Kln⁡(1+pkσk2)s.t.∑k=1Kpk−P≤0−pk≤0,   k=1,2,…,K(3.1)\begin{aligned}&\min_{p_1, \dots, p_K} - \sum^K_{k = 1}\ln\left( 1 + \frac{p_k}{\sigma^2_k}\right)\\ \mathrm{s.t.}&\sum^K_{k = 1}p_k - P \le 0\\ &-p_k \le 0,\ \ \ k = 1, 2, \dots, K \end{aligned}\tag{3.1}s.t.​p1​,…,pK​min​−k=1∑K​ln(1+σk2​pk​​)k=1∑K​pk​−P≤0−pk​≤0,   k=1,2,…,K​(3.1)

   −ln⁡(⋅)−\ln(·)−ln(⋅)是一个凸（convex）函数，即目标函数是一个凸函数，同样的，也不难证明可行域 (p1,p2,…,pK)∈P(p_1, p_2, \dots, p_K)\in \mathcal{P}(p1​,p2​,…,pK​)∈P 是凸的。即该优化问题是一个凸优化问题，拉格朗日函数为：

L(p1,p2,…,pK;λ0,λ1,…,λK)=−∑k=1Kln⁡(1+pkσk2)+λ0(∑k=1Kpk−P)−∑k=1Kλkpk(3.2)\begin{aligned} &\mathcal{L}(p_1, p_2, \dots, p_K; \lambda_0, \lambda_1, \dots, \lambda_K) \\ &= -\sum^K_{k = 1} \ln \left(1 + \frac{p_k}{\sigma^2_k}\right) + \lambda_0 \left( \sum^K_{k = 1} p_k - P \right) - \sum^K_{k = 1}\lambda_k p_k \end{aligned} \tag{3.2}​L(p1​,p2​,…,pK​;λ0​,λ1​,…,λK​)=−k=1∑K​ln(1+σk2​pk​​)+λ0​(k=1∑K​pk​−P)−k=1∑K​λk​pk​​(3.2)

∂L∂pk=−1σk2+pk+λ0−λk=0,   k=1,2,…,K(3.3a)\frac{\partial \mathcal{L}}{\partial p_k} = - \frac{1}{\sigma^2_k + p_k} + \lambda_0 - \lambda_k = 0, \ \ \ k = 1, 2, \dots, K \tag{3.3a}∂pk​∂L​=−σk2​+pk​1​+λ0​−λk​=0,   k=1,2,…,K(3.3a)

λ0(∑k=1Kpk−P)=0(3.3d)\lambda_0 \left( \sum^K_{k = 1}p_k - P\right) = 0\tag{3.3d}λ0​(k=1∑K​pk​−P)=0(3.3d)

  式 (3.3a)(3.3\mathrm{a})(3.3a) 可以重写为：

λk=λ0−1σk2+pkpk=1λ0−λk−σk2(3.4)\begin{aligned}\lambda_k = \lambda_0 - \frac{1}{\sigma^2_k + p_k} \\ p_k = \frac{1}{\lambda_0 - \lambda_k} - \sigma^2_k \end{aligned}\tag{3.4}λk​=λ0​−σk2​+pk​1​pk​=λ0​−λk​1​−σk2​​(3.4)

  如果 ∑k=1Kpk<P\sum^K_{k=1} p_k < P∑k=1K​pk​<P ，则 λ0=0,λk=−1/(σk2+pk)<0\lambda_0 = 0, \lambda_k = -1/(\sigma^2_k + p_k)<0λ0​=0,λk​=−1/(σk2​+pk​)<0 ，与 (3.3b)(3.3\mathrm{b})(3.3b) 矛盾。故 ∑k=1Kpk=P\sum^K_{k = 1} p_k = P∑k=1K​pk​=P ，由 pkp_kpk​ 非负性可知 pkp_kpk​ 不全为零。

  不妨假设 σ12≤σ22≤⋯≤σK2\sigma^2_1 \le \sigma^2_2 \le \dots \le \sigma^2_Kσ12​≤σ22​≤⋯≤σK2​ ，因为 pkp_kpk​ 不可能全为零，对应的 λkλ_kλk​ 为零，对于这些 pk>0,λk=0p_k > 0, λ_k = 0pk​>0,λk​=0 的信道：

pk=1λ0−λk−σk2=1λ0−σk2(3.5)p_k = \frac{1}{\lambda_0 - \lambda_k} - \sigma^2_k = \frac{1}{\lambda_0} - \sigma^2_k \tag{3.5}pk​=λ0​−λk​1​−σk2​=λ0​1​−σk2​(3.5)

  如果 λk>0λ_k > 0λk​>0 ，则 pk=0p_k = 0pk​=0 ，有：

λk=λ0−1σk2+pk=λ0−1σk2(3.6)\lambda_k = \lambda_0 - \frac{1}{\sigma^2_k + p_k} = \lambda_0 - \frac{1}{\sigma^2_k} \tag{3.6}λk​=λ0​−σk2​+pk​1​=λ0​−σk2​1​(3.6)

  由于 σk2σ^2_kσk2​ 单调递增，则有：

λK≥λK−1≥⋯≥λk>0,   pK=pK−1=⋯=pk=0(3.7)\lambda_K \ge \lambda_{K-1} \ge \dots \ge \lambda_k > 0,\ \ \ p_K = p_{K-1} = \dots = p_k = 0 \tag{3.7}λK​≥λK−1​≥⋯≥λk​>0,   pK​=pK−1​=⋯=pk​=0(3.7)

  即上文所提到的，对于好的信道（λk=0λ_k = 0λk​=0），分配功率 pkp_kpk​ 满足 pk+σk2=1/λ0p_k +σ^2_k = 1/\lambda_0pk​+σk2​=1/λ0​ 为定值，对于差的信道（λk>0\lambda_k > 0λk​>0，应当注意的是 (3.7)(3.7)(3.7) 指出，如果噪声功率为 σk\sigma_kσk​ 的信道是差信道，则噪声功率更大的信道也是差信道），则不分配功率，直到功率分配尽为止。1/λ01 / \lambda_01/λ0​ 即上文的 p⋆p^⋆p⋆。

连续并联信道的注水定理

  对于某一变换域（如频域上）的有色噪声 σ2(f)σ^2 (f)σ2(f)，注水问题的优化可以写作：
